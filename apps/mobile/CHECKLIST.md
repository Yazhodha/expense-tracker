## Common Issues & Quick Fixes

**Issue**: `npx expo prebuild` fails with Firebase error
**Fix**: Make sure config files are in `apps/mobile/` directory

**Issue**: Google Sign-In fails
**Fix**: Check `.env` has correct Web client ID (not iOS or Android ID)

**Issue**: App crashes on launch
**Fix**: Run `npx expo prebuild --clean` and try again

**Issue**: Can't find Firebase config
**Fix**: Verify files are named exactly:
- `google-services.json` (lowercase, no spaces)
- `GoogleService-Info.plist` (exact capitalization)
